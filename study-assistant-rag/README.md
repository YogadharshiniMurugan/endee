# 📚 AI Study Assistant using RAG with Endee

## 🚀 Project Overview

This project is a Retrieval-Augmented Generation (RAG) based AI assistant that uses Endee as a vector database to perform semantic search and answer user queries.

## 🎯 Objective

To build an AI/ML application using Endee’s dataset and vector search capabilities to demonstrate a real-world use case.

## 🔥 Features

* Semantic search using embeddings
* Retrieval-Augmented Generation (RAG)
* Fast similarity search using vector database
* Simple command-line interface

## ⚙️ How It Works

1. User enters a query
2. Query is converted into vector embeddings
3. Endee vector database retrieves similar data
4. Most relevant result is returned as the answer

## 🛠️ Technologies Used

* Python
* Sentence Transformers
* NumPy
* Endee (Vector Database)

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```

## 📦 Use of Endee

* Stores embeddings of dataset
* Performs similarity search
* Enables efficient retrieval for RAG system
